# Пишем асинхронную карточную игру в браузере

В какой-то момент я захотел составить конспект самых необходимых
знаний, которые нужны для разработки современного веб-приложения.
Вашему вниманию представляется этот конспект-руководство по
созданию своего веб-приложения.

В этом руководстве мы разберем большое количество инструментов,
которые можно использовать в современной веб-разработке на примере
собственной карточной игры.

Данное руководство составлено как список упрощающих разработку рецептов.
О рецептах, они:

- подкреплены объяснением от автора;
- содержат субъективное мнение и не претендуют на абсолютную верность;
- обильно приправлены ссылками на документацию;
- можно ими пользоваться, а можно и не пользоваться.
